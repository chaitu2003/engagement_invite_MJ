# Jagadeesh & Moukthika Engagement Invitation

A modern, lightweight Next.js App Router invitation website with a premium ribbon-opening intro, optimized images, elegant typography, smooth scroll sections, and optional background music.

## Features

- Next.js App Router
- Tailwind CSS
- Minimal Framer Motion animations
- Ribbon unwrap opener screen
- Jagadeesh & Moukthika names
- ANNAM & CHEKKA families invitation text
- Engagement date: 10th May 2026
- Background music support
- Mobile-first responsive layout
- Vercel-ready deployment

## Add background music

Place your audio file here:

```txt
public/music/song.mp3
```

Keep it small, preferably below 2 MB. Browsers allow music playback only after user interaction, so music is controlled from the site after the invitation opens.

## Add gallery images

Replace the images in:

```txt
public/gallery/
```

Use compressed `.webp` or optimized `.jpg` images for best Lighthouse performance.

## Run locally

```bash
npm install
npm run dev
```

Open:

```txt
http://localhost:3000
```

## Build

```bash
npm run build
npm run start
```

## Deploy on Vercel

Push this project to GitHub and import it into Vercel.
